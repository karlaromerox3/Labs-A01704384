Cuestionario 1: BD vd DBMS

¿Qué es entonces una base de datos - DB?

Es una colección de datos organizados que normalmente está organizada en tablas para hacer el procesamiento y busqueda de datos mucho más eficiente. [1]

¿En qué casos es conveniente usar bases de datos?

  • Cuando hay archivos interrelacionados

  • Cuando la información la requieren varios usuarios

  • Cuando se necesita tener reportes y busquedas dinámicas de información. [2]

¿Qué es un sistema de gestión de base de datos - DBMS?

Es el conjunto de una base de datos y una colección de programas y aplicaciones para acceder a ellos, los cuales den un entorno conveniente y eficiente para extraer y almacenar la información.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Responsabilidades:

Interacción con el gestor de archivos

Obtener, registrar y eliminar registros de archivos.

Asegurar mínima redundancia de datos

No duplicar la información y los formatos de manera que la información sea consistente en todos los archivos

Implantación de seguridad

Tener seguridad de que si el servicio se cae o tiene accesos no permitidos, los datos siguen existiendo en alguna parte y no recibirán ningún daño


Funciones:

Administración del almacenamiento y diccionario de datos

Crear estructuras que ayuden a validaciones, relaciones de los datos; además de los datos requeridos para acceder a ellos.

Presentación de datos

Significa poder pasar del código de BD a una interfaz para que el usuario interactúe con todos los archivos.

Control de Concurrencia

Asegurar que si hay varios usuarios accediendo a la misma información, todos tengan la versión más actualizada y coordinada.
