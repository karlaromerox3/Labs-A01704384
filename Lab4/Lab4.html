¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Las principales diferencias vienen en que Java es un lenguaje que se usa para aplicaciones de Android, aplicaciones de escritorio y para empresas, mientras que JavaScript se usa para añadir interactividad a las páginas web como una alternativa de flash. Java se corre en una máquina virtual y JavaScript en un navegador web. [1]

Las similitudes radican en que ambos soportan el paradigma orientado a objetos y tienen sintaxis parecida.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

Date.now(): Da la hora y fecha actual

Date.parse(): Transforma la fecha y devuelve los milisegundos desde 1970 hasta la fecha.

Date.getTimezoneOffset(): Devuelve la diferencia entre el horario UTC y el local

Date.getUTCDate(): Devuelve la fecha del tiempo universal.

Date.setUTCDate(): Define la fecha de acuerdo al horario UTC.[2]

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)?

concat(): Une dos o más arreglos y los regresa en uno nuevo.

copyWithin(): Copia una parte del arreglo y lo pega en otra ubicación del mismo.

filter(): Crea un nuevo arreglo con los elementos que pasan una función testing.

reduce(): Reduce un arreglo en un solo valor de izquierda a derecha. Es util para invertir números que llegan como String

unshift(): Añade nuevos elementos al inicio de un array y recorre el resto; devuelve el array resultante. [3]

¿Cómo se declara una variable con alcance local dentro de una función?

var permite definir una variable dentro de una función y su alcance. Ya que resultó confuso, se introdujo let que hace que una variable exista en el bloque en el que existe.[4]

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Una variable global puede ser usada en todo el archivo de JS, por lo que va a ser facilmente modificada en cualquier función, además de que no puede haber ninguna otra que se llame igual dentro de cualquier bloque o función.[5]

¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?

Match porque toma una cadena y busca en un archivo todas sus coincidencias para devolverlas. Puede servir en una applicación web para implementar una barra de búsqueda en el sitio, para artículos y contenido. [6]
