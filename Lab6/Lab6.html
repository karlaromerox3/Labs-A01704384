¿Cuáles son las diferencias entre los posibles valores de la propiedad position?

static Los objetos static usan el flujo hacia abajo de la página, y no tiene en cuenta los valores de top, right, bottom y left
relative Permite posicionar con respecto al flujo normal pero si hace caso a top, right, bottom y left
absolute Acepta la lista de valores antes dicha pero no el flujo normal, toma su posición con respecto al elemento más cercano si es que es padre.
fixed está fuera del flujo y acepta top, bottom, etc. La diferencia es que solo toman como referencia la ventana del ordenador y es fijo respecto al scroll
inherit Permite heredar del elemento padre
initial es lo mismo que no usar ningún atributo o el static
sticky Es como si usaras relative hasta que se desliza y funciona como fixed, normalmente se usa para menús.

¿Qué es el zIndex y para qué sirve?

Indica el orden de un elemento cuando estos se superponen. Entre mayor sea el z-index, más arriba está la capa. Vale la pena mencionar que no es heredable, y solo aplica para elementos posicionados. [3]

¿Cuáles son los valores estándar para la propiedad visibility?

Visible, hidden, collapse(solo para tablas) y inherit (herencia de elementos padre) [2]
